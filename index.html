What it checks

Eleven signals, one verdict

All probes are rootless — no admin password, no privacy entitlements. The app uses only what macOS exposes to any regular user.

Wi-Fi crypto
Encryption type (WPA2 / WPA3 / open / older) + signal strength.
Internet reach
Captive portal detection via Apple's well-known hotspot probe.
Public IP
Your apparent IP, city, country, ISP — so you know what the rest of the internet sees.
DNS resolver
Identifies the resolver (Cloudflare / Google / Quad9 / router) and flags unfamiliar ones.
Download
A ~5 MB test download, with a fallback chain so it works almost everywhere. Skippable for metered networks.
Response time
Latency + jitter to 1.1.1.1 and the gateway. The jitter number is what tells you whether calls will stutter.
Safety check
Looks at the local ARP table for duplicate MACs and locally-administered gateway MAC — both signs of interception or a rogue access point.
Path
A short traceroute — how many hops your traffic takes to reach the public internet.
VPN
Detects when the default route is a tunnel interface (utun, ipsec, ppp).
Private Relay
Reports whether Apple iCloud+ Private Relay is reachable on this network.
LAN devices
Discovers every device on your subnet and labels each with type and brand — phones, computers, TVs, speakers, printers, IoT.
Source

Open, on GitHub

Python backend (FastAPI + rumps menu-bar + WKWebView), bundled as a native .app via py2app. No telemetry, no analytics, no network calls beyond what the probes need.
